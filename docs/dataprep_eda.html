Data Collection Method

Data was collected from Counter-Strike 2 (CS2) professional match demos using the demoparser2 Python library. This library allows parsing of CS2 demo files (.dem) to extract detailed game statistics and events.

Raw Data Sources

  • Demo Files: Professional CS2 matches from HLTV.org
  • Match Coverage: 3 complete professional matches (8 maps total)
  • Match IDs:
    • 2385025_99903 (semperfi vs underground)
    • 2385383_99901 (aether vs full-send)
    • 2385385_99900 (incontrol vs nocturnal)

üßπ Data Cleaning Process

Data Quality Assessment

Initial analysis revealed several data quality issues that needed addressing:

Issues Identified:
  • Missing Values: 71% missing in assist data (expected in CS:GO)
  • Data Types: Boolean columns stored as objects
  • Outliers: Extreme distance values from parsing errors
  • Standardization: Inconsistent categorical variables
Cleaning Steps Applied:
  • Handle Missing Values: Fill assists with "No Assist" (common in CS:GO)
  • Type Conversion: Convert boolean and numeric columns
  • Outlier Removal: Remove extreme distance values using IQR method
  • Standardization: Normalize team names and weapon categories
